autoedit.github.io
Automatic photo editing using deep neural networks
This site uses a deep neural network to apply content-aware color transformations to your photos. All computation is done locally on your device using Onnx.js.
Some examples of the results produced by the network
View more results produced by the algorithm or select your own image below. Note that in order to increase speed and decrease memory usage, images will be downscaled to a maximum size of 960x960 before processing. Does not currently support Safari, iOS, or Firefox for Android due to limitations of onnx.js. It is recommended that you select an image which has not already been edited for optimal results.
